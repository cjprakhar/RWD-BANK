# Bank Portal Project

A modern, full-stack banking dashboard application featuring:
- **Secure Login/Signup**
- **Real-time Transaction History**
- **Interactive Balance Charts**
- **Dark/Light Theme Support**
- **Persistent Data Storage**

## How to Run

You will need two terminal windows open to run the full application.

### Terminal 1: Backend Server
This runs the API and database.
```bash
node server.js
```
*Server will start on port 3001.*

### Terminal 2: Frontend Application
This runs the React user interface.
```bash
npm run dev
```
*Application will be accessible at http://localhost:5173*

## Build for Production
To create an optimized build:
```bash
npm run build
```
